## condicion usando if

```python
numero1=1
numero2=2
if numero1 < numero2:
    print("numero1 es menor")
else:
    print("numero1 es mayor o igual")
```

# comparaciones

* == igualdad
* != no son iguales
* < menor a
* > mayor a
* <= menor o igual
* >= mayor o igual
